Computational estimation of the impact of mutations on protein-ligand affinity based on Alphafold and MD simulation

University of Pisa

Code

To align structure predicted by Alphafold with the crystallographic structure from Protein Data Bank

Install chimera and run this script with the command:

python3 alignStructure.py -f [input folder] -o [output folder] -c [path to chimera run file] -p + [path to the crystallographic structure]

The input folder is a folder contains subfolder, each subfolder is a result folder from AlphaFold. Inside each subfolder there should be a file name ranked_0.pdb

Path to chimera run file often ends with "bin/chimera"
